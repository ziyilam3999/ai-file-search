To configure Confluence integration:

  1. Copy .env.example to .env
  2. Set CONFLUENCE_URL (e.g., https://company.atlassian.net)
  3. Set CONFLUENCE_EMAIL (your Atlassian email)
  4. Set CONFLUENCE_API_TOKEN (Generate API Token)
  5. Restart the application
